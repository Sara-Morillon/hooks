# useCopy

`useCopy` offers helpers to handle clipboard copy.

## Example

```tsx
import { useCopy } from '@saramorillon/hooks'

function MyComponent() {
  const [{ loading, error }, copy] = useCopy()

  if (loading) return <Spinner />
  if (error) return <Error />

  return (
    <button onClick={() => copy('Something')}>
      Copy
    </button>
  )
}
```

## API

```typescript
useCopy(): ICopyStatus
```

### Arguments

None

### Returns

An array containing:

- an object containing:
  - `loading: boolean` - indicates weither the copy is currently pending or not.
  - `error?: unknown` - the error potentially generated by the copy.
- `copy: (data: string | ClipboardItems) => void` - the copy function.
